## ğŸ› ï¸ Getting Started

### Prerequisites
- Basic computer knowledge
- Text editor (VS Code recommended)
- Web browser (Chrome/Firefox)
- Node.js (for backend development)
- MySQL (for database work)
- MongoDB (for NoSQL database)
- Git (for version control)

### Setup Instructions
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Uttamsaini15/Web-Devlopment.git
    cd Web-development
    ```

2.  **Frontend Setup**:
    - Open HTML files directly in browser
    - Use Live Server extension for development

3.  **Backend Setup**:
    ```bash
    cd "Web-Development/Back-End/NodeJS"
    npm install
    node script.js

    ```  

4.  **Database Setup**:
    - Install MySQL and MongoDB
    - Run SQL scripts from the SQL directory
    - See `Web-Development/MongoDB/mongodb-basics.js` for basic MongoDB commands.
